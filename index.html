Floyd-Warshall algorithm

G is an nxn adjacency matrix

for k in range(n):
  for i in range(n):
    for j in range(n):
      G[i][j] = min(G[i][j], G[i][k] + G[k][j])

Floyd-Warshall algorithm

Vectorized

for k in range(n):
  n_n = G[:, k].reshape(n, 1) + G[k, :].reshape(1, n)
  G = np.minimum(G, n_n)
for k in range(n):
  for i in range(n):
    for j in range(n):
      G[i][j] = min(G[i][j], G[i][k] + G[k][j])

  • for_python << for_go < k_cpu = for_java < for_c < k_gpu
